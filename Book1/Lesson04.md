# 第4課　部屋に　机と　いすが　あります

## 上

### 本课重点

1. 存在句：“（某处）有（某物）” “（某物）在（某处）”
2. 并列助词“と”：“~和~”
3. 语气助词”ね“：表确认
4. “も”+否定：表全面否定
